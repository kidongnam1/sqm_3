# 동시 편집(여러 AI·사람) 시 권장

Cursor·IDE는 **파일 잠금을 자동으로 서로에게 공유하지 않습니다.**

1. 같은 파일을 두 세션이 같이 건드리면 **Git 충돌**이나 **한쪽 덮어쓰기**가 날 수 있습니다.
2. **권장:** 큰 JS·HTML을 고칠 땐 한 시점에 **담당 1명(또는 1에이전트)만** 수정.
3. 병행이 필요하면 **다른 파일**로 나누거나, **브랜치**를 갈라서 나중에 merge 하세요.
4. 수동으로 막고 싶다면 작업 전에 팀/채팅에 “`sqm-inline.js` 수정 중”만 알려도 혼선이 크게 줄어듭니다.

(이 문서는 참고용 — 자동 락 기능은 없습니다.)
