## 사용 가이드 (Template Notes)

### 본 템플릿을 사용하는 경우
1. 위 모든 `<...>` 플레이스홀더를 실제 값으로 치환
2. 작업 지시서 파일명: `SQM_WORK_ORDER_YYYY-MM-DD.md` (날짜만 변경)
3. 프로젝트 루트에 저장
4. 대표님 승인 후 시공 시작

### 패치 종류별 추천 프로필
- **소규모 (1 파일, 1 함수)**: P1만 + Q1 (테스트만), 30분
- **중규모 (3~5 파일)**: P1 + P2 + Q1 + E2E, 60~90분
- **대규모 (전체 모듈)**: P1~P5 + Q1+Q2 + E2E + 회귀 테스트, 120~180분
- **긴급 핫픽스**: P1만 + Manual smoke, 15분

### 안전 원칙
- 모든 패치 전: `pre-<patchname>-<DATE>` 태그
- 모든 패치 후: `post-<patchname>-<DATE>` 태그
- 모든 파일 변경 시: `.bak.<YYYYMMDD>` 백업
- 자동화 절대 금지 영역: 사용자 데이터 (`exports/`, `data/db/`, `backup/`)
